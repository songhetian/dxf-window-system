# Moser Glass Pro 功能介绍

本系统是一款专为玻璃及门窗行业设计的桌面价格计算工具，旨在通过“基础材料 -> 组合产品 -> 报价清单”的逻辑链条，实现快速、精准的自动化报价。

## 1. 核心功能模块

### 1.1 报价中心 (Project Manager)
这是用户停留时间最长的模块，主要逻辑如下：
- **实时计算引擎**：当用户选择一个“组合产品”并输入尺寸（宽/高）时，系统会立即根据预设的公式（面积或周长）计算出该项的零售标价和内部价格。
- **价格类型切换**：支持一键在“零售标价”和“内部价格”之间切换，方便用户在面对客户或内部核算时灵活处理。
- **清单累加逻辑**：用户可以将不同规格、不同型号的产品“添加到项目”，系统会自动汇总总面积、总金额以及各项明细。
- **自动化报表**：基于 `xlsx` 库，一键将当前屏幕上的所有报价明细导出为标准的 Excel 电子表格。

### 1.2 材料配置 (Database Center)
系统的“大脑”，负责定义报价的底层逻辑：
- **组合产品模型**：允许将多个“基础材料”绑定在一起。例如，一个“三玻两腔”产品可能由 3 层玻璃、2 个隔条和 1 项加工工艺组成。
- **分级计算体系**：
    - **基础材料**：维护最底层的单价。
    - **组合产品**：继承材料单价，并定义计价方式（按平米或按米）。
- **材料分类治理**：通过标签化管理，确保数以百计的材料能被快速检索。
- **费率与加价比例**：支持设置全局的额外费用（如管理费、运输费、加工费比例），这些费用会自动累加到最终的总价中。

### 1.3 系统设置 (System Settings)
- **数据安全保障**：基于本地 SQLite 数据库，提供完整的导出备份和还原功能，确保数据不会丢失。
- **外观定制**：内置深色/浅色模式切换，适配不同办公环境。

## 2. 价格计算逻辑
- **面积计价**：`宽度(m) * 高度(m) * (各材料单价之和) * 数量`
- **周长计价**：`(宽度(m) + 高度(m)) * 2 * (各材料单价之和) * 数量`
- **最终总价**：`(各产品项金额之和) * (1 + 额外费率比例)`
